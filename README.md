# Campus Parking

Aplicación web de gestión de parqueadero con CRUD de tipos de vehículo, servicios de ingreso/salida y perfil de usuario.

## Cómo ejecutar
1. Abrir `DOCSHTML/index.html` en el navegador.
2. Usar la pestaña **Servicios** para registrar ingresos y finalizar servicios.
3. Usar la pestaña **Tipos** para registrar y administrar tipos de vehículos.
4. El botón **Perfil** carga un modal con datos de usuario guardados en `localStorage`.

## Estructura principal
- `DOCSHTML/index.html` - Página principal del panel.
- `DOCSHTML/login.html` - Pantalla de inicio de sesión.
- `ESTILOSCSS/` - Estilos globales, componentes y responsividad.
- `FUNCIONESJS/` - Lógica de la app: controladores, CRUD y cálculos.
- `PAGINAS/` - Vistas dinámicas cargadas en el `main`.
